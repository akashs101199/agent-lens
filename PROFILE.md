<img src="https://img.shields.io/badge/Open_Source-MIT-green?style=for-the-badge" />
<img src="https://img.shields.io/badge/TypeScript-Production--Ready-blue?style=for-the-badge" />

<img src="https://img.shields.io/badge/Tests-268%2B-brightgreen?style=for-the-badge" />

**AgentLens** is an open-source TypeScript library that provides complete observability into AI agent execution. It automatically logs every LLM call, tool invocation, reasoning step, and error in a structured, queryable format.

### 🎯 What It Solves
AI agents are black boxes. You can't see:
- What decisions the agent made and why
- Which tools it called and their execution time
- How much API calls cost
- Whether it made mistakes and recovered

### ✨ What Makes It Different
- **Framework agnostic** (works with Anthropic, OpenAI, or any SDK)
- **Privacy-first** (PII detection & redaction before logging)
- **Zero dependencies** in core package
- **Beautiful UI** (colored terminal output + JSONL for AI tools)
- **Production-ready** (268 tests, 4000+ LOC, zero TypeScript errors)

### 📊 Built Completely with Claude Code
- **Claude Haiku** — Rapid prototyping
- **Claude Sonnet** — Main implementation
- **Claude Opus** — Architecture decisions
- **Replit AI Agent** — Generated 70% of code
- **Result**: 6 packages, full test suite, comprehensive docs in 2 weeks
